# C++ Learning Simulator with Turbo IDE
This is an easy-to-use C++ programming simulator designed for beginners. I created this to help my classmate learn both the Turbo IDE interface and C++ programming fundamentals in a simplified, guided environment.

Live Demo: [[Demo Here]](https://turbo-simulator.vercel.app/)

## Quick Start Guide
### Prerequisites
Make sure you have installed:
- Node.js (v18 or higher)
- npm/bun or yarn

### Installation
Clone the repository:
```sh
git clone https://github.com/anfastech/turbo-ide-simulator
cd turbo-ide-simulator
```
